使用說明：
輸入WASD按鍵移動Hero移動，以H代表，在遊戲內還有其他物件：
	1. Creature: 踩到會對角色進行battle，進行猜拳遊戲，如果輸了Hero會被扣HP。
		- Creature1: Bat, 以 B 表示
		- Creature2: Wolf, 以 W 表示
		- Creature3: Skeleton, 以 S 表示
	   如果贏過Creature，則會對Creature進行刪除，代表該Creature不會在版面上出現

	2. Creature的不同行為：
		- Bat：可以穿牆
		- Wolf：不可穿牆
		- Skeleton：不可穿牆
		
	3. Item: 給予Hero不同效果，例如加經驗、回血，或是傳送
		- Trigger:給予角色經驗 
		- Meat：給予角色血量HP
		- Portal (傳送門)：傳送hero到另一個傳送門

	4. 檔案存讀檔功能
